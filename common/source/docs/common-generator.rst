==========
Generators
==========

Support for generators is currently available in master.  Within ArduPilot the term generator is used to broadly group all devices that convert some form or fuel into electrical power.  Hence, the list of currently supported generators includes hydrogen fuel cells.

The following are all currently supported within ArduPilot:
